Erase entire flash?

This will permanently erase all 16 MB of flash memory on the ESP32. Firmware, Wi‑Fi credentials, saved settings, and route cache will be deleted.

The board will not run FlightScnr until you flash firmware again.

Before you start

  1. Use Chrome on a desktop computer or laptop.
  2. Connect the LILYGO T-Encoder Pro board to your computer with a USB cable.
  3. If the USB port does not appear: push the screen down (BOOT) and hold, tap RESET on the back of the board, then click Connect.
  4. If Connect or Install fails: hold the screen pressed in (BOOT), click Connect or Install again, and release once flashing begins.
  5. Do not hold the screen in while plugging in USB - it is the BOOT button (GPIO 0) and can enter download mode unintentionally.

Choose Full install for a first-time install, after Erase flash, or when replacing factory firmware (flashes at 0x0; clears Wi‑Fi and saved settings). Update app only flashes the application at 0x10000 and keeps settings when bootloader and partitions already match. Upload .bin uses the same offsets from the filename (merged → 0x0, otherwise 0x10000).
